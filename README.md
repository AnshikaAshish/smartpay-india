# smartpay-india
AI-Powered Fintech Microservices Platform built with Spring Boot + Claude AI

Generate complete User entity and UserDTO 
for SmartPay India with fields:
- id, fullName, email, password, phone
- Aadhaar number (masked)
- UPI ID
- accountStatus, createdAt, updatedAt
- Use Java 17, Spring Boot 3.x, Lombok, JPA annotations
